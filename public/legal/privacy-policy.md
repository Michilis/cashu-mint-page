# Privacy Policy

**Effective Date: December 2024**  
**Last Updated: December 2024**

Welcome to Cashumints.space ("we," "our," or "us"). This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you visit our website and use our services. Please read this privacy policy carefully. If you do not agree with the terms of this privacy policy, please do not access the site.

## 1. Information We Collect

### 1.1 Information We Do NOT Collect
Cashumints.space is designed with privacy-by-design principles. We do not collect:
- Personal identifiable information (names, email addresses, phone numbers)
- User accounts or registration data
- Payment information or financial data
- Location data or device fingerprinting
- Cookies for tracking purposes
- Personal browsing history or behavioral profiles

### 1.2 Information We DO Collect

#### Analytics
- We use Plausible Analytics, a privacy-focused service
- Data collected includes: page URLs, HTTP referrers, browser type/version, operating system, device type, and location (country/region/city)
- No cookies, personal data, or individual user tracking
- IP addresses are hashed and discarded daily - never stored
- Analytics data is publicly accessible at: https://analytics.azzamo.net/cashumints.space
- All data is aggregated and anonymous

#### Technical Information
- Server logs for security and performance monitoring
- IP addresses (temporarily logged and automatically deleted)
- Basic browser information for compatibility
- Error logs for debugging purposes

#### Public Nostr Data
- Nostr public keys when users submit reviews
- Review content published to public Nostr relays (decentralized servers)
- Timestamps of public interactions
- Cryptographic signatures of published content
- Note: Nostr is not a blockchain - data is stored on independent relay servers

## 4. Data Security

### 4.1 Security Measures
We implement appropriate security measures including:

- Secure server configurations

- No storage of sensitive personal data

### 4.2 Data Retention
